# Canvas 通用 Agent 助手设计

Date: 2026-07-07

## 目标

把当前画布通用 Agent 从“只会产 CanvasPlan 的入口”升级为一个先理解用户、再选择行动方式的助手。它必须能自然聊天，也能在明确画布意图时调用当前画布专业能力完成节点、连线、计划与任务执行。

## 用户体验

- 用户发送 `hi`、`你好`、`在吗` 时，助手应立即自然回复，并简短说明可以聊天、总结信息、分析需求或操作当前画布。
- 用户只是泛聊或问普通问题时，助手直接回答；如果问题依赖最新信息，进入联网搜索总结分支。
- 用户提出“设计当前系统能力”“帮我规划这个功能”一类请求时，助手先分析需求，只问一个最关键的问题，确认后形成明确计划。
- 用户明确要求创建、修改、连接或运行画布内容时，助手读取当前画布上下文，输出安全的 CanvasPlan 或请求必要确认。
- 助手不得因为低信号输入沉默；不能安全执行时必须返回可见的澄清或能力缺口说明。

## 意图路由

新增或扩展 `intent-analysis`，输出稳定分类：

| 分类 | 触发 | 行为 |
| :--- | :--- | :--- |
| `smallTalk` | 问候、寒暄 | 返回 `answer`，不创建画布节点 |
| `generalChat` | 普通聊天、概念解释、非画布问题 | 返回 `answer` |
| `searchSummary` | 包含“最新、今天、查一下、搜索、新闻、价格”等时效性诉求 | 调用联网搜索能力后总结；能力不可用时返回明确缺口 |
| `requirementPlanning` | 设计系统能力、复杂产品需求、实现方案讨论 | 先分析需求并提出澄清问题或计划 |
| `canvasOperation` | 创建节点、连线、运行、生成图片/视频、编排工作流 | 进入 CanvasPlan/工具执行分支 |
| `clarify` | 核心目标无法判断 | 只问一个关键问题 |

## 执行分支

### 1. 聊天分支

`smallTalk` 和 `generalChat` 返回 `AgentAnswerResponse`。默认 planner 必须至少覆盖问候、日期时间、普通知识类短问，避免没有模型或没有网关时沉默。

### 2. 联网总结分支

`searchSummary` 需要一个受控搜索能力，优先作为 ToolRuntime 工具暴露，例如 `web.search` 或等价工具。输出应包含摘要、关键事实和来源链接。搜索失败或未配置时，返回 `clarification` 或 `answer`，说明当前缺少联网搜索能力，不得假装搜索过。

### 3. 需求分析分支

`requirementPlanning` 不直接改画布。流程为：

1. 复述目标与已知约束。
2. 找出缺口。
3. 只问一个最关键问题；若信息足够，输出计划。
4. 用户确认后再进入 CanvasPlan 或开发执行。

### 4. 画布专业操作分支

`canvasOperation` 使用当前已有契约：

- `CanvasPlan.kind` 只能是 `plan` 或 `clarify`。
- 节点类型、连接规则、运行动作必须通过共享真源校验。
- 生图/生视频仍走入队和 IPC 终态，不同步阻塞。
- 自动执行必须受 UI 开关、Agent policy 和权限策略共同约束。

## 前端展示

现有 `ChatPanel` 和 `CanvasChatBox` 继续作为入口。两者统一展示：

- 普通回答。
- 澄清问题。
- 联网总结来源。
- PlanCard。
- 工具调用进度和权限请求。
- 失败原因。

## 错误处理

- 任何分支失败都要发出 `agent.responseReady` 或可见错误消息。
- `job.failed` 不能是唯一终态；前端应能看到“为什么没有回答”。
- 搜索、模型、工具、计划清洗失败分别使用稳定 `errorClass`。

## 测试策略

- `intent-analysis` 单元测试覆盖问候、泛聊、联网触发词、需求规划、画布操作。
- orchestrator 测试覆盖 `hi/你好` 返回 `answer`。
- IPC 测试覆盖非 CanvasPlan 回复必须 emit `agent.responseReady`。
- UI 测试覆盖画布浮窗收到 `answer` 后显示文本且不调用 `getCanvasPlan`。
- 搜索能力先以接口/工具 mock 测试，避免真实网络进入单元测试。

## 非目标

- 不在 CanvasPlan 中加入可执行代码。
- 不绕过当前连接矩阵和 Plan sanitizer。
- 不把普通聊天强行变成画布节点。
- 不在无搜索能力时伪造“联网结果”。
